  ## Imprimindo e capturando valores no console
  Console.WriteLine()
   - printa e quebra linha
  
  Console.Write()
   - apenas printa

  Console.ReadLine();
   - Esse método e suas variáveis abre um input para o usuário.

  Console.Clear();
   - limpa o console
